Ember

Ember includes a built-in set of trusted JavaScript libraries. The LLM can request any of the following libraries by using its alias in the frontmatter of a code block.

d3 (D3.js)
three (Three.js)
p5 (p5.js)
anime (Anime.js)
chartjs (Chart.js)
matter (Matter.js)

Example usage in a `javascript-live` block: 

---
libs:
  - three
  - anime
---
// Your code here...
