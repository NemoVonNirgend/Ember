Ember

Ember includes a built-in set of trusted JavaScript libraries. To create an interactive element, use a standard `javascript` code block and add a frontmatter section (`---`) at the top to request libraries.

d3 (D3.js)
three (Three.js)
p5 (p5.js)
anime (Anime.js)
chartjs (Chart.js)
matter (Matter.js)

Example usage in a `javascript` block: 

---
libs:
  - three
  - anime
---
// Your code here...
// It will be given a 'root' element to append content to.
const renderer = new THREE.WebGLRenderer();
root.appendChild(renderer.domElement);
// ...etc
